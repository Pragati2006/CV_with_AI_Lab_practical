# Computer Vision with Artificial Intelligence Lab

This repository contains a collection of practical implementations covering core concepts in Data Processing, Machine Learning, Computer Vision, and Web Scraping. Each experiment demonstrates real-world applications using Python and commonly used libraries.

---

## Lab Objectives

* Understand data preprocessing, visualization, and wrangling techniques
* Apply machine learning algorithms to real-world datasets
* Explore computer vision fundamentals and object detection
* Implement web scraping techniques for data collection

---

## Experiments Overview

### 1. Data Processing вЂ“ Most Specific Hypothesis

* Implemented Candidate Elimination (Find-S Algorithm)
* Reads training data from a `.csv` file
* Outputs the most specific hypothesis consistent with training data

---

### 2. Data Visualization вЂ“ Hypothesis Space

* Visual representation of all hypotheses consistent with training examples
* Helps understand version space boundaries

---

### 3. Data Wrangling

* Reading and cleaning datasets from `.csv` files
* Handling missing values and formatting issues
* Preparing data for analysis and modeling

---

### 4. COVID-19 Data Analysis

* Real-world dataset used for:

  * Data processing and wrangling
  * Visualization using Choropleth World Map
  * Plotting active, recovered, and death cases
* Libraries used:

  * pandas, plotly, matplotlib

---

### 5. Linear Regression вЂ“ Salary Prediction

* Predict salary based on experience
* Reads dataset from `.csv`
* Implements model training, prediction, and visualization

---

### 6. KNN вЂ“ Iris Classification

* Classification using K-Nearest Neighbors (KNN)
* Dataset: Iris dataset from `.csv`
* Includes preprocessing, training, and evaluation

---

### 7. Web Scraping

* Extract product data from shopping websites (e.g., Amazon)
* Uses requests and BeautifulSoup
* Extracts product names, prices, and ratings

---

### 8. Image Processing Basics

* Color conversions: RGB to BGR and BGR to Grayscale
* Creating black and white images
* Drawing shapes such as lines, rectangles, and circles

---

### 9. Face / Eye / Smile Detection

* Implemented using Haar Cascade Classifiers
* Detects faces, eyes, and smiles using OpenCV

---

### 10. Decision Tree Classifier

* Applied on a dataset of choice
* Includes preprocessing, training, prediction, and evaluation
